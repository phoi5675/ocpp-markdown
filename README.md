# ocpp-markdown

OCPP document converted to markdown

## Convert asciidoc to pdf using asciidoc-pdf

```bash
asciidoctor-pdf --theme ../styles/themes/theme.yml -a pdf-fontsdir="../styles/fonts;GEM_FONTS_DIR" ${asciidoc_file}
```
